# Quero Landing Page

A pixel-perfect, production-ready landing page for Quero - a comprehensive mock test platform for NEET UG, NEET PG & other entrance exams.

## 🚀 Features

- **Modern SaaS Startup Aesthetic** - Clean, professional design
- **Fully Responsive** - Mobile, tablet, and desktop optimized
- **Smooth Animations** - Built with Framer Motion
- **Performance Optimized** - Fast loading and rendering
- **Accessibility** - Semantic HTML and WCAG compliant
- **SEO Ready** - Optimized metadata and semantic markup

## 📦 Tech Stack

- **React** - UI library
- **Vite** - Next-generation frontend build tool
- **Framer Motion** - Animation library
- **Lucide React** - Icon library
- **Pure CSS** - No Tailwind CSS
- **Inter Font** - Google Fonts

## 📋 Components

- **Navbar** - Responsive navigation with dropdown menus
- **Hero** - Eye-catching hero section with CTAs
- **ExamCards** - Interactive exam cards
- **Stats** - Trust statistics section
- **Features** - Detailed feature showcase with phone mockup
- **Footer** - Comprehensive footer with links

## 🛠️ Installation

### Prerequisites
- Node.js (v14+)
- npm or yarn

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/shubhamrk021-droid/quero.git
   cd quero
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
   The site will open at `http://localhost:5173`

4. **Build for production**
   ```bash
   npm run build
   ```

5. **Preview the production build**
   ```bash
   npm run preview
   ```
